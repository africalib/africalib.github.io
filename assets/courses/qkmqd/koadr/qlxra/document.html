웹 브라우저로 웹 서버에 HTTP 요청을 하려면 어떻게 해야 할까요? 아래 중 하나를 하면 됩니다.

1. 웹 사이트 접속: 웹 브라우저에 URL을 직접 입력해서 웹 애플리케이션에 접속하면 웹 서버와 HTTP 통신을 수행할 수 있습니다. 예를 들어, 네이버 URL(http://naver.com)로 접속해서 HTTP 요청을 하고, HTTP 응답 결과를 출력하는 것 입니다.

2. 자바스크립트 코드 실행: 웹 브라우저가 자바스크립트 코드를 실행함으로써 웹 서버와 HTTP 통신을 수행할 수 있습니 다. 예를 들면, Fetch API를 활용해 게시글 데이터가 있는 서버에 HTTP 요청을 하고, HTTP 응답 결과를 출력하는 것 입니다.


둘 중에 자바스크립트 코드를 실행해 HTTP 통신을 수행하는 방법을 알아보겠습니다
