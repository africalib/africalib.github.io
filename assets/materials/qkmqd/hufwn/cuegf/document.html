마이크로소프트는 윈도우 운영체제에 인터넷 익스플로러(Internet Explorer)를 기본 웹 브라우저로 탑재하기 시작했습니다. 덕분에 익스플로러의 점유율은 급격히 상승했고, 전 세계 사용률의 95%를 차지할 정도가 되었죠. 당시 웹 개발자들은 대부분 익스플로러에 맞춰 개발해도 큰 문제가 없었습니다.


하지만 시간이 지나 다양한 웹 브라우저가 등장했음에도, 익스플로러는 웹 표준을 무시하고 독단적인 행보를 이어갔습니다. 이 때문에 개발자들은 크로스 브라우징(cross browsing) 문제를 겪게 되었죠. 즉, 같은 코드가 브라우저마다 다르게 동작하는 상황이 발생한 겁니다. 이로 인해 익스플로러는 개발자들 사이에서 ‘공공의 적’처럼 여겨졌습니다.


그러던 2008년, 구글이 크롬(Chrome)이라는 웹 브라우저를 출시했습니다. 크롬은 빠르고 가벼운 웹 경험을 제공했을 뿐만 아니라, 웹 표준을 준수하고 강력한 개발자 도구를 지원해 개발자들로부터 큰 호응을 얻었습니다. 시간이 흐르면서 크롬은 전 세계에서 가장 많이 사용되는 웹 브라우저가 되었고, 익스플로러는 점점 사용률이 줄어 2022년 6월 15일 공식 지원이 종료되었습니다.


이제 구글 크롬이 주도하는 시대가 열렸습니다. 이번 챕터에서는 크롬을 비롯한 최신 웹 브라우저를 설치하고, 프론트엔드 개발 환경을 세팅하는 방법을 알아보겠습니다.
